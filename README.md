# user-info-intenttech
a full-stack application that manages users information.

> Create.

> Edit.

> View.

## Built with React, TypeScript, Tailwind and NestJS

## Install

```sh
cd .\backend\
npm install

cd .\frontend\
npm install
```

## Usage

add .env to backend

```sh
cd .\backend\
npm start
cd .\frontend\
npm start
```

Spins up local servers on port 3000 and 4000


Example:

```sh
Local:     http://localhost:3000
```
